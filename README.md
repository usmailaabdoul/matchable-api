## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# DB migration
$ npx prisma migrate dev

# Run custom seed command to populate DB
$ npm run seed

# To start the dev development
$ npm run start

# production mode
$ npm run start:prod
```
